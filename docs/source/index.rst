Welcome to MARLBS's documentation!
===================================

**MARLBS** is a Python Project for AI training and large scale warfare lovers
that creates two teams that fight each other on episodes and improve by using Reinforcement Learning.
It achieves this by using skorch <https://skorch.readthedocs.io/en/stable/>`_
and offers a *simple* and *intuitive* interface for the user.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
